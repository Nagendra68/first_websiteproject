How to become a successful Web Developer

Step 1

Learn a language

Select any one coding language and learn all it's basics.

Step 2

Learn DSA

Learn DSA concepts and practice 20-25 questions on each topic.

Step 3

Learn Development

Learn HTML CSS JS Nodejs and any one framework such as Angular,React.

Step 4

Make Projects

Make projects which will showcase your learning of development.

Step 5

Deploy projects

Deploy your projects so that anybody can see it by clicking on your link.

Step 6

Learn git and github

Learn about Git,Github and Open source contributions.

Step 7

Make a resume

Make a resume showcasing all your skills,learning and projects.

Step 8

Apply for job

Find companies having vacancies from linkedin,naukri.com etc and apply for job requiring skills that you have.
